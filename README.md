## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

This project uses the Google Maps API to map the job centers obtained from the following link https://datos.gob.mx/busca/dataset/catalogo-de-centros-de-trabajo-2022. A form was created to filter these centers in a more specific way.

## Getting Started

This is an example of how you may give instructions on setting up your project locally.
To get a local copy up and running follow these simple example steps.

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.
* The format of the excel file should be .csv.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Get a free API Key at [API google](https://developers.google.com/maps/documentation/javascript?hl=es-419)
2. Clone the repo
   ```sh
   git clone https://github.com/J0harAR/CCTmaps.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Enter your API in `index.html`
   ```js
   <script src="https://maps.googleapis.com/maps/api/js?key='ENTER YOUR API'"></script>
   ```
5. Enter your name of the file in `app.js`
   ```js
   fs.createReadStream(' enter the name of the .csv file')
   ```

## Usage

 Use the following command to start the project in console.
   ```sh
   nodemon app.js 3000
   ```
 Choose the parameters you want and click on the "mapear" button and it will give you as a result the points of the job centers.
